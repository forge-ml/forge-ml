# Frequently Asked Questions

## OpenAI API Key Requirement

Forge ML utilizes OpenAI's language models to process and generate structured data based on user-defined schemas. The API key:

- Authenticates and secures user interactions with OpenAI services
- Ensures users are billed only for their specific usage
- Allows users to manage their own API usage limits and billing directly through OpenAI

## Caching Types

| Common Caching | Individual Caching |
|----------------|---------------------|
| Shared across multiple users, processes, or systems | Specific to a single user, process, or system |
| Implemented at a higher level (e.g., server or network) | Local and not shared with others |
| Serves multiple clients | Serves a single client |
